## Featured Projects

### MedScribe

Clinical handwriting OCR that converts handwritten prescriptions into structured medication records using transformer-based OCR with a clinician review workflow.

**Tech:** `Python` `FastAPI` `PyTorch` `TrOCR` `React`

---

### ECG Arrhythmia Classifier

Interpretable ECG screening system using signal processing and machine learning with leakage-free validation.

**Tech:** `Python` `Flask` `React` `scikit-learn`

---

### Silent Swap

Research study on adversarial errors in clinical NLP — swapping drug names for plausible SNOMED siblings and measuring where a five-stage pipeline first catches them. Written up as a full paper.

**Tech:** `Python` `PyTorch` `Bio_ClinicalBERT` `SNOMED CT`

---

### Sahi

AI-powered nutrition assistant that analyzes foods and provides personalized recommendations for people managing chronic conditions.

**Tech:** `React Native` `Expo` `TypeScript` `Supabase`

---

### SkillBridge

Career intelligence platform that recommends learning paths through skill-gap analysis and personality-based matching.

**Tech:** `React` `Node.js` `Express` `MongoDB`

---

### Synthetic Health Data Audit

Privacy-preserving synthetic healthcare data generation using VAE and CTGAN, evaluated for fidelity, utility, and privacy.

**Tech:** `Python` `PyTorch` `CTGAN`
